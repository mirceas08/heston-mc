heston-mc
=========
Monte Carlo simulation for the Heston stochastic volatility model using various discretization schemes.
Code written for the assignment in the [Advanced Derivatives](http://didattica.unibocconi.it/ts/tsn_anteprima2006.php?cod_ins=20245&anno=2015&IdPag=5862) class at Bocconi University. For info and references check the docs.
